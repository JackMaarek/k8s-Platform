# Azure modules

Not yet implemented. Placeholder for future multi-cloud support.

Expected modules:
- `aks/`           — AKS cluster (equivalent of aws/eks)
- `pod-identity/`  — Azure AD Pod Identity / Workload Identity (equivalent of aws/irsa)
- `nodepool/`      — AKS node pools (equivalent of aws/nodegroup)
- `key-vault/`     — Azure Key Vault (equivalent of aws/secrets-manager)
- `vnet/`          — Virtual Network (equivalent of aws/vpc)
