# GCP modules

Not yet implemented. Placeholder for future multi-cloud support.

Expected modules:
- `gke/`           — GKE cluster (equivalent of aws/eks)
- `workload-identity/` — Workload Identity for pod-level GCP auth (equivalent of aws/irsa)
- `nodepool/`      — GKE node pools (equivalent of aws/nodegroup)
- `secret-manager/` — GCP Secret Manager (equivalent of aws/secrets-manager)
- `vpc/`           — VPC network (equivalent of aws/vpc)
